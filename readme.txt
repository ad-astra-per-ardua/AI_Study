GitHub 저장소에 파일 업로드 

cmd 창열고

1. 깃 저장소로 쓸 파일위치 들어가기 cd 위치

2 .저장소 만들기
	 git clone 깃주소

3.저장소에 올릴 파일 넣어놓기

4.추가 및 커밋

git add 파일명

git commit -m "메시지아무거나" 

5.푸시하기
git push origin main
